Painter

Below you can find list of default keys uses in application. You can change it in presentation-draw-settings.json file in your %APPDATA% folder.

Tools:

  • 1 - select / manage
  • 2 - paint
  • 3 - rectangle
  • 4 - line
  • 5 - arrow
  • 6 - text

In select mode you can group / ungroup objects with Ctrl + G and Ctrl + Shift + G

Colors:

  • r - red
  • g - green
  • b - blue
  • p - pink
  • y - yellow
  • o - orange

Other stuff:

  • delete - delete selected objects in selection mode
  • [ - decrease brush size. You can do this with mouse wheel.
  • ] - increase brush size. You can do this with mouse wheel.
  • c - clear screen
  • / - toggle white board
